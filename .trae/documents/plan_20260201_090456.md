## 修复看板任务卡片中责任人显示问题

**问题**: 看板任务卡片(`createTaskElement`方法)缺少责任人显示逻辑,导致配置了责任人的任务不显示责任人信息。

**修复方案**:
在 `src/components/ProjectKanbanView.ts` 文件的 `createTaskElement` 方法中,在第7992行 `taskContentContainer.appendChild(titleContainer);` 之后,添加责任人显示代码:

```typescript
// 添加责任人显示
if (task.assigneeId) {
    const assigneeEl = document.createElement('div');
    assigneeEl.className = 'kanban-task-assignee';
    assigneeEl.style.cssText = `
        font-size: 12px;
        color: var(--b3-theme-on-surface-light);
        margin-top: 6px;
        display: flex;
        align-items: center;
        gap: 4px;
    `;
    const personName = this.personManager.getPersonName(task.assigneeId);
    if (personName) {
        assigneeEl.innerHTML = `<svg style="width: 14px; height: 14px; opacity: 0.7;"><use xlink:href="#iconUser"></use></svg><span>${personName}</span>`;
        taskContentContainer.appendChild(assigneeEl);
    }
}
```

**验证**: 修复后,配置了责任人的任务卡片会在标题下方显示责任人图标和姓名。

**部署**: 修改完成后重新构建插件并部署。
